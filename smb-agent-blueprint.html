FAQ

Common Questions

Do I need to know how to code?
No. The blueprint is built for operators and business owners. The templates use configuration files (YAML/Markdown) that are filled in like forms, not coded from scratch. If you can use Notion or Airtable, you can deploy these agents.
Which AI model does this work with?
All of them. The blueprint is deliberately model-agnostic — Claude, GPT-4o, Gemini, Mistral, or local models via Ollama. You swap the LLM provider in one config line. Your workflow doesn't break when a better model launches.
How long does deployment actually take?
The first agent can be running in 2–4 hours following the playbook. A full 3–5 agent back-office workforce typically takes 3–5 days of setup. There's no ongoing development — once configured, agents run autonomously.
What if the agent templates don't match my tools?
The integration maps cover 12 platforms and include a generic webhook/API pattern that works with any modern SaaS tool. The blueprint's Tool Integration module walks through adapting any template to any platform in under an hour.
Is there a subscription or recurring cost?
Zero. One payment, instant download, yours forever. The only recurring cost you'll have is your LLM API usage — which the 3x Rule Calculator will show you is a rounding error compared to the labor savings.
